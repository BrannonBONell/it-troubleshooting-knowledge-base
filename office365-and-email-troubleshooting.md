# Office 365 and Email Troubleshooting

# What is Microsoft 365?

Microsoft 365 (formerly Office 365) is a cloud productivity platform.

Common services:
- Outlook
- Teams
- OneDrive
- Exchange Online
- SharePoint

---

# Common Email Problems

- unable to send email
- unable to receive email
- Outlook not opening
- mailbox full
- login failures
- sync issues

---

# Initial Troubleshooting Steps

1. Verify internet connectivity
2. Confirm account credentials
3. Restart Outlook
4. Test Outlook Web Access
5. Check Microsoft service status

---

# Outlook Troubleshooting

# Restart Outlook

Simple restarts often resolve:
- freezing
- sync issues
- authentication prompts

---

# Safe Mode

Launch Outlook in safe mode:

```powershell
outlook.exe /safe
```

Useful for:
- add-in conflicts
- startup crashes

---

# Create New Outlook Profile

Corrupted profiles can cause:
- crashes
- sync failures
- connection problems

---

# Common Scenarios

# Scenario 1
User cannot receive email.

Possible causes:
- mailbox full
- sync issue
- server outage
- spam filtering

---

# Scenario 2
Outlook repeatedly asks for password.

Possible causes:
- expired credentials
- MFA issues
- corrupted credentials cache

---

# Scenario 3
Teams not loading.

Possible causes:
- corrupted cache
- network issue
- outdated client

---

# Basic Troubleshooting Mindset

1. Verify credentials
2. Test web access
3. Isolate local vs cloud issue
4. Check service health
5. Document findings

---
