# Home Lab Setup and Virtualization

# What is a Home Lab?

A home lab is a personal learning environment used to practice IT skills.

Common uses:
- Linux practice
- networking
- Active Directory
- virtualization
- cloud labs

---

# Virtualization Basics

Virtualization allows multiple operating systems to run on one physical computer.

Common software:
- VirtualBox
- VMware Workstation
- Hyper-V

---

# Example Home Lab Setup

Host System:
- Windows 11

Virtual Machines:
- Ubuntu Linux VM
- Windows Server VM
- Windows 10 VM

---

# Skills Practiced

- Linux administration
- networking
- Active Directory
- troubleshooting
- remote access

# Basic VirtualBox Setup

1. Install VirtualBox
2. Download ISO file
3. Create VM
4. Allocate RAM/storage
5. Install operating system

# Networking Modes

# NAT
VM shares host internet connection.

---

# Bridged Adapter
VM appears as separate device on network.

Useful for:
- server testing
- networking labs

# Why Home Labs Matter

Home labs demonstrate:
- initiative
- hands-on learning
- troubleshooting experience
