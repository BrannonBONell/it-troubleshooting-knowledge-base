# PowerShell Basics and Administration

# What is PowerShell?

PowerShell is Microsoft's command-line and scripting platform.

Used heavily in:
- system administration
- cloud management
- Active Directory
- automation
- enterprise IT

---

# Opening PowerShell

Press:

```text
Windows + X
```

Select:
- Windows PowerShell
- Terminal

---

# Basic Commands

# Current Directory

```powershell
Get-Location
```

---

# List Files

```powershell
Get-ChildItem
```

---

# Display Processes

```powershell
Get-Process
```

---

# Display Services

```powershell
Get-Service
```

---

# Stop Process

```powershell
Stop-Process -Name notepad
```

---

# Networking Commands

# Test Connectivity

```powershell
Test-Connection google.com
```

---

# IP Configuration

```powershell
ipconfig
```

---

# Useful Administration Tasks

# Restart Computer

```powershell
Restart-Computer
```

---

# Check System Information

```powershell
Get-ComputerInfo
```

---

# Event Logs

```powershell
Get-EventLog -LogName System
```

---

# Why PowerShell Matters

PowerShell skills are valuable for:
- cloud support
- Windows administration
- DevOps
- system administration
- automation

Basic scripting knowledge helps automate repetitive IT tasks.
