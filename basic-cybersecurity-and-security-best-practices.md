# Basic Cybersecurity and Security Best Practices

# Introduction

Cybersecurity focuses on protecting:
- systems
- accounts
- networks
- applications
- data

Basic security knowledge is important for all IT roles.

---

# Common Threats

- phishing
- malware
- ransomware
- password attacks
- social engineering

---

# Phishing

Phishing attempts trick users into revealing:
- passwords
- MFA codes
- sensitive information

Common indicators:
- suspicious links
- urgent language
- unexpected attachments

---

# Multi-Factor Authentication (MFA)

MFA adds additional verification beyond passwords.

Examples:
- authentication apps
- text codes
- hardware tokens

MFA significantly improves security.

---

# Password Best Practices

- use long passwords
- avoid password reuse
- use password managers
- enable MFA

---

# Principle of Least Privilege

Users should only have access required for their role.

Reduces:
- accidental damage
- security exposure

---

# Common Security Practices

- keep software updated
- avoid suspicious downloads
- lock devices when unattended
- verify email senders
- monitor login alerts

---

# Malware Basics

Malware types:
- viruses
- trojans
- ransomware
- spyware

Common symptoms:
- slow performance
- popups
- unusual network activity

---

# Basic Incident Response

1. Identify issue
2. Isolate affected system
3. Report incident
4. Document findings
5. Follow company procedures

---

# Why Cybersecurity Matters

Security awareness is valuable in:
- help desk
- cloud support
- system administration
- DevOps
- AI infrastructure
