# NVMe Troubleshooting Guide

## Symptoms
- Drive not detected in BIOS
- Windows installer cannot find SSD
- Random disconnects

## Possible Causes
- Faulty motherboard slot
- Loose mounting
- PCIe lane conflicts
- BIOS configuration
- Defective SSD

## Troubleshooting Steps
1. Reseat SSD
2. Test alternate M.2 slot
3. Update BIOS
4. Disable CSM
5. Check PCIe configuration
6. Test drive in another system
